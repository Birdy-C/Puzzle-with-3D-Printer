# Astrobot Clock

The **Astrobot Clock** normally switches between displaying the time and an Astrobot eye. When touched, it changes to a heart-shaped eye as a fun interactive feature.

## Components Used

- **Arduino Nano**  
- **Mini USB Cable**  
- **Timer Module**  
- **Vibration Sensor Module**  
- **1.8-inch TFT Screen**
